# Templo dos Enigmas em C
## História do jogo
Você acorda dentro de um templo antigo, cercado por símbolos misteriosos e tochas acesas.
Na sua frente existe um portal fechado por três enigmas. Para escapar do templo, você precisa resolver todos os enigmas
corretamente.
## Objetivo do jogo
O objetivo do jogo é resolver os três enigmas e abrir o portal de saída.
## Tema
Fuga, desbloqueio e enigmas.
## Linguagem utilizada
C
## Funcionalidades
- Menu interativo
- História inicial
- Enigma com número secreto
- Enigma com palavra-chave
- Enigma com sequência lógica
- Controle de progresso do jogador
- Sistema de vitória ou derrota
## Como executar
Para compilar o programa, use:
```bash
gcc main.c -o templo-enigmas
```
Para executar no Linux ou Mac:
```bash
./templo-enigmas
```
Para executar no Windows:
```bash
templo-enigmas.exe
```
